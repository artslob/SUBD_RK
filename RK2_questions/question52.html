Question: 52

You can use the EM Support Workbench for which of the following tasks? (Choose all correct answers.)

Answer: B,C,D,E

Explanation: B, C, D, and E are correct. You can use the Support Workbench for viewing problems and incidents, manually running health checks, generating additional dumps and test cases for Oracle Support, creating and tracking a service request to Oracle Support, collecting all data related to a problem and packaging it, and closing the problem when it has been resolved. A is incorrect because the EM Support Workbench cannot run recommended repairs. However, you can initiate advisors that can recommend repairs.
