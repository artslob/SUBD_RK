Question: 58

Which of the following database objects are not candidates for RMAN image copies? (Choose all that apply.)

Answer: B,C,E

Explanation: B, C, and E cannot be backed up as image copies. You should never back up online redo log files under any circumstances, you do not need to back up flashback logs, and password files cannot be backed up by RMAN. A is incorrect because any datafile can be backed up as an RMAN image copy. D is incorrect because you can back up archived redo log files as image copies.
