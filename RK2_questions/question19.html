Question: 19

Your database is not using an RMAN recovery catalog. Which of the following commands are notavailable to you in this scenario? (Choose all that apply.)

Answer: B,D

Explanation:B and D are correct. If you are using the control file only for RMAN backups, backup informationmay age out of the control file and therefore will not be kept forever. In addition, schema changesare stored only in a recovery catalog but not in the control file.A is incorrect because you can configure RMAN parameters whether or not a recovery catalog isconfigured.C is incorrect because you can always delete backups that are older or outside the definedretention policy.
