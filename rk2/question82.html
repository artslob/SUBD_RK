With the DBMS_STATS package you can view and modify optimizer statistics gathered for database objects. The Oracle RDBMS allows you to collect statistics of many different kinds as an aid to improve performance. This package is concerned with optimizer statistics only. Given that Oracle sets automatic statistics collection of this kind by default, this package is intended for only specialized cases. The statistics of interest to be viewed or modified can reside in the dictionary or in a table created in the user's schema for this purpose.
